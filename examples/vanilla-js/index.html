This example shows how to integrate Bug Reporter with a vanilla JavaScript/HTML project using a CDN.

Demo Actions

Try these actions to generate different types of logs and errors for testing:

Integration Code

<!-- Include via CDN -->
<script src="https://unpkg.com/@zudsniper/bugs@latest/dist/index.js"></script>
<link rel="stylesheet" href="https://unpkg.com/@zudsniper/bugs@latest/dist/styles.css">

<script>
// Initialize bug reporter
const { BugReporter, defaultConfigs } = window.ZudsniperBugs;

// Secure configuration using proxy mode
const config = {
  integrations: {
    webhook: {
      mode: 'proxy',
      endpoint: '/api/bug-report/webhook'
    }
  },
  ui: {
    title: 'Report an Issue',
    theme: 'auto',
    position: 'bottom-right'
  }
};

// Create and mount bug reporter
const container = document.createElement('div');
document.body.appendChild(container);

ReactDOM.render(
  React.createElement(BugReporter, { config }),
  container
);
</script>

Note: This is a simplified example. In a real implementation, you would configure your actual webhook URL or integration service.
